# C++ Transactional-Banking-System(SQLite)

## Features
- Create Account
- Deposit Money
- Withdraw Money
- Transfer Money (with transaction safety)
- View Account Details
- Transaction History
- Delete the Account

## Tech Stack
- C++
- SQLite (embedded database)
- SQL Transactions

## Concepts Used
- OOP (Classes, Encapsulation)
- File/Database Handling
- Prepared Statements (SQL)
- Transactions (BEGIN / COMMIT / ROLLBACK)

## How to Run
1. Compile:
   ./ulinux.sh

2. Run:
   ./bank

## Future Improvements
- GUI
